# How to create project with Expo

## Create Project (Blank)
```bash
$ expo init expo-blank
#choose blank [enter]
#change expo -> name [Teachdev]
#change expo -> slug [teachdev]
#Yarn v1.15.2 found. [Y]
#..waiting
$ cd expo-blank
$ yarn start
```

## Create Project (Tabs)
```bash
$ expo init expo-tabs
#choose tabs [enter]
#change expo -> name [Expo Tabs]
#change expo -> slug [expo-tabs]
#Yarn v1.15.2 found. [Y]
#..waiting
$ cd expo-tabs
$ yarn start
```
