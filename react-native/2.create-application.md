## Create application with React Native

### Begin create project
```bash
$ cd <root-directory>
$ create-react-native-app kiddeepass-superadmin
# ...waiting 
```

### Test
```bash
$ cd kiddeepass-superadmin
$ yarn start | yarn
```

### Fixed
```bash
# Unable to start server
$ sudo sysctl -w kern.maxfiles=5424880

```
