## Move settings or configure of Express with middleware
```bash
$ mkdir libs
$ cd libs
$ nano middlewares.js
module.exports = app => {
    app.set("port", 8080);
    app.set("json spaces", 4);
};
```

## Loading middlewares.js to bootstrap
```bash
...
// remove configure zone
consign()
    .include("models")
    .then("libs/middlewares.js")
    .then("routes")
    .into(app);
...
```

## Create libs/boot.js
```bash
$ nano libs/boot.js
module.exports = app => {
    app.listen(app.get("port"), () => {
        console.log(`API - Port ${app.get("port")}`);
    });
};
```

## Loading boot.js to bootstrap
```bash
...
consign()
    .include("models")
    .then("libs/middlewares.js")
    .then("routes")
    .then("libs/boot.js")
    .into(app);
...
// remove running zone
```
