# How to install PHP71 via manaul

### Install PHP71 with Brew
```bash
$ brew search php71
$ brew install php71
# brew reinstall -s php71
```

### Start
```bash
$ brew services start homebrew/php/php71
$ brew services stop homebrew/php/php71
$ brew services restart homebrew/php/php71
```

## Version
```bash
$ php -v
# PHP 7.1.14
```
