# Installation tools for micronaut 

## SDKMAN
Go to https://sdkman.io
```bash
$ sdk
# zsh: command not found: sdk
$ curl -s "https://get.sdkman.io" | bash
# close your terminal and open again
$ sdk
# .....
```

## Maven
```bash
$ sdk install maven
```

## Micronaut
```bash
$ mn
# zsh: command not found: mn
$ sdk install micronaut
```
