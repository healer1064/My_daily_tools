# How to handling basic configurations with debug

## How to do it #1
```bash
app = Flask(__name__)
app.config['DEBUG'] = True
```

## How to do it #2
```bash
...
app.debug = True
```

## How to do it #3
```bash
...
app.run(debug=True)
```

## How to do it #4
```bash
DEBUG = True
TESTING = True
app = Flask(__name__)
app.config.from_object(__name__)
```

## How to do it #5 (object)
```bash
...
app.config.from_object('myapplication.default_settings')
```
