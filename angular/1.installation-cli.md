# Install Angular CLI & Tools

## Node
```bash
$ node -v
# v8.6.0
$ npm -v
# 5.5.1
```

## Check Angular CLI
```bash
$ ng -v
# zsh: command not found: ng
```

## Installation Angular CLI
```bash
$ npm install -g @angular/cli
$ ...waiting...
$ ng -v
# @angular/cli: 1.4.7
```

## Configuration package manager
```bash
$ ng set --global packageManager=yarn
```
