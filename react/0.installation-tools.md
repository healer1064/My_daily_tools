# How to installation tools for React

## Check first
```bash
$ create-react-app
# zsh: command not found: create-react-app
```

## Installation CLI
```bash
$ npm install -g create-react-app
```

## Uninstall CLI
```bash
$ npm uninstall -g create-react-app
```

## Current!!!! Using without install CLI
```bash
$ npx create-react-app ced-client
$ cd ced-client
```
