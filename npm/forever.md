# Running backgroud process

## Installation
```bash
$ sudo npm install -g forever
```

## List
```bash
$ sudo forever list
```

## Start
```bash
$ sudo forever start <script>
# $ sudo forever start -c "yarn start" .
```

## Stop
```bash
$ sudo forever list ***getting uid of script
$ sudo forever stop <uid>
```

## Restart
```bash
$ sudo forever list ***getting uid of script
$ sudo forever restart <uid>
```
