# How to installation Strapi

## Installation Development

```bash
git clone https://github.com/strapi/strapi.git
# ...waiting
cd strapi
npm install -g yarn
npm run setup
```

## Installation CLI

```bash
npm install strapi@beta -g
# + strapi@3.0.0-beta.15
# added 430 packages from 297 contributors in 55.741s
```
