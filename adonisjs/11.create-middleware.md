# How to create middleware with Adonis

## Create middleware
```bash
$ cd /var/www/adonis-api
$ adonis make:middleware CurrencySelector
# App/Middleware/CurrencySelector
```

## Register middleware
```bash
$ nano start/kernel.js
```
```bash
...
const globalMiddleware = [
    ...
    'App/Middleware/CurrencySelecor',
]
...
```
