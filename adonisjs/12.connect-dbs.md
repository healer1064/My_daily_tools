# How to connect databases with Adonis

## Install the package
```bash
$ cd /var/www/adonis-api
$ adonis install @adonisjs/lucid
# config/database.js
```

## Configure
```bash
$ nano start/app.js
```
```bash
...
const providers = [
    ...
    "@adonisjs/lucid/providers/LucidProvider",
]
...
const aceProviders = [
    ...
    "@adonisjs/lucid/providers/MigrationsProvider",
]
...
```
