# How to create project with Adonis

## New Project

```bash
adonis new adonis-first
```

## Start Project

```bash
cd adonis-first
adonis serve --dev
# Look at your browser 127.0.0.1:3333
```

## New Project (Slim)

```bash
adonis new adonis-slim --slim
cd adonis-slim
adonis serve --dev
```

## New Project (API)

```bash
adonis new adonis-api --api-only
cd adonis-api
adonis serve --dev
```
