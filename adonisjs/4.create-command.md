# How to create command with Adonis

## Create
```bash
$ cd /var/www/adonis-api
$ adonis make:command SendReminderCommand
# app/Commands/SendReminder.js
```

## Running
```bash
$ node ace
```

## Running without node command
```bash
$ chmod +x ace
```

```bash
#!/usr/bin/env node
```

```bash
$ ./ace
```
