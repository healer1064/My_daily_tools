# How is the main.go file organized

## Your code

```bash
package main
```

## Structure

```bash
package main        #(1)Package declaration

import "fmt"        #(2)Import other packages that we need

func main() {        #(3)Deciare functions, tell Go to do things
    fmt.PrintLn("Hi there!")
}
```
