# How to update theme for iterm2

## Installation theme with curl
```bash
$ cd
$ sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```

## Update theme
```bash
$ cd 
$ nano ~/.bash_profile
```

```bash
#zsh
ZSH_THEME="agnoster"
```

```bash
$ source ~/.bash_profile
```
